# Asset Brief

## Direction

`Lumorix: Null District` should read as dark, premium, mysterious and slightly dangerous. The first public pitch should look like a co-op investigation game, not a generic pixel platformer.

Use strong silhouettes, limited animation, sharp UI screens, eerie signal effects and memorable interactables. The placeholder rectangles should be replaced in the order below.

## Pixel Art Priority 1: Playable Beta Identity

- Player base body: `Signal Runner`, 32x48 or 48x64, idle/run/jump/fall/dash/stun/pulse/death.
- Player variant body: `Relay Warden`, heavier coat/helmet silhouette.
- Player variant body: `Null Analyst`, scanner/hood/cable silhouette.
- Face/helmet readable 1-bit highlight pass for dark backgrounds.
- Remote player tint variants for multiplayer readability.
- Small cosmetic skin overlay system: coat, mask, glow line, backpack.

## Pixel Art Priority 2: Case 001 Objects

- Signal Haven shelter background tiles.
- Signal Haven extraction gate, idle and active animation.
- Null Market terminal, safe and non-pay-to-win visual.
- Shelter terminal.
- District Entrance broken street tiles.
- First Relay machine, locked/solving/solved states.
- Evidence Fragment pickup, idle pulse animation.
- Underground Sector A tunnel tiles.
- Broken Terminal, corrupted/decoded states.
- Relay Core pickup, strong silhouette.
- Door/gate sprites for area transitions.

## Pixel Art Priority 3: Threats And Atmosphere

- Corrupted Scout: patrol/chase/stun/collapse.
- Signal Wraith: floating/stalking/flicker/disperse.
- Enemy projectile signal bolt.
- Environmental hazard: unstable static field.
- Environmental hazard: leaking relay arc.
- Shadow foreground silhouettes for each map.
- Parallax skyline/background ruins.
- Animated monitor glyphs and warning panels.
- Glitch particles: small cyan/green/pink fragments.

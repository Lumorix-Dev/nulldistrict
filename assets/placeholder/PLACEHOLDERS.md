# Placeholder Asset Policy

The playable beta generates simple pixel-block Phaser textures at runtime so the game can be played immediately.
Replace them with real pixel art by keeping stable asset keys:

- `player`
- `remote-player`
- `enemy-corrupted-scout`
- `enemy-signal-wraith`
- `terminal`
- `relay`
- `pickup`
- `door`
- `slash`
- `projectile`

Drop final sprites into `assets/sprites`, tiles into `assets/tilesets`, UI icons into `assets/ui/icons`, and audio into `assets/audio`.

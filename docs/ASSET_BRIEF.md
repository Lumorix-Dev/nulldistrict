# Asset Brief

## Direction

`Lumorix: Null District` should read as dark, premium, mysterious and slightly dangerous. The first public pitch should look like a co-op investigation game, not a generic pixel platformer.

Use strong silhouettes, limited animation, sharp UI screens, eerie signal effects and memorable interactables. The placeholder rectangles should be replaced in the order below.

## Pixel Art Priority 1: Playable Beta Identity

- Player base body: `Signal Runner`, 32x48 or 48x64, idle/run/jump/fall/dash/stun/pulse/death.
- Player variant body: `Relay Warden`, heavier coat/helmet silhouette.
- Player variant body: `Null Analyst`, scanner/hood/cable silhouette.
- Face/helmet readable 1-bit highlight pass for dark backgrounds.
- Remote player tint variants for multiplayer readability.
- Small cosmetic skin overlay system: coat, mask, glow line, backpack.

- West Sync Node and East Sync Node, idle/armed/solved states with matching silhouettes.
